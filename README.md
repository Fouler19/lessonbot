# Телеграм-бот для онлайн-школы

## Описание
Этот бот создан для студентов онлайн-школы. Он помогает быстро получить расписание уроков через Telegram.

## Возможности
- приветствие пользователя;
- интерактивные кнопки;
- просмотр расписания уроков;
- раздел "О боте";
- раздел "Помощь";
- мотивирующее сообщение.

## Структура проекта
school_schedule_bot/
- main.py
- config.py
- keyboards.py
- schedule_data.py
- requirements.txt
- README.md

## Используемые технологии
- Python
- aiogram
- Telegram Bot API

## Как запустить
1. Создать бота через BotFather
2. Скопировать токен
3. Вставить токен в файл `config.py`
4. Установить зависимости:

```bash
pip install -r requirements.txt
